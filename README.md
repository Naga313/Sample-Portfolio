# Sample-Portfolio
A simple, responsive sample portfolio built with HTML, CSS, and JavaScript. It showcases a personal header, navigation, portfolio highlights, education, community involvement, peer reviews, and a footer. It also includes a dark mode toggle for quick theming. This is intended as a sample portfolio for demonstration and learning purposes.

# Features

Clean, responsive layout suitable for desktop and mobile.

Simple top navigation for Home, Projects, Blogs, and Contact.

“Portfolio Highlights” section listing sample components like responsive layouts, storefronts, interactive widgets, and landing pages.

“Career Achievements” with brief role summary and a link placeholder to LinkedIn.

“Community Involvement” with a link placeholder to GitHub.

“Education” card with key specializations (e.g., System Analysis, Web Accessibility, Performance).

“Peer Reviews” section with three short testimonials.

Footer with current year and author name.

Dark Mode toggle button.

# Tech Stack

HTML5 for structure

CSS3 for styling and layout

JavaScript for interactions (e.g., dark mode)

Project Structure

index.html – main page markup

style.css

script.js

# Accessibility Notes

Used semantic HTML for sections and headings.

Ensured sufficient color contrast in both light and dark modes.
